# Sampa Project

## Requirements
 - [Docker](https://docs.docker.com/docker-for-mac/)

## Config
- Copy the .env.dist file to .env
- Set the environment variables values at the .env file corresponding your local enviroment (or leave as is if following this doc)
- Set the environment variable `GOOGLE_MAPS_KEY` with your Google Maps API Key

## Run on your terminal

```
docker-compose up
```

```
open http://localhost:8080
```
